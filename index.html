Java
Es un lenguaje de programación de alto nivel y orientado a objetos. Fue creado por James Gosling y su equipo en Sun Microsystems en la década de 1990. Java se diseñó para ser un lenguaje de programación portable, lo que significa que el código escrito en Java puede ejecutarse en diferentes plataformas sin necesidad de reescribirlo.
Html
Es el lenguaje utilizado para definir la estructura y el contenido de una página web, mientras que CSS se encarga del diseño visual y JavaScript agrega interactividad y funcionalidad a la página. Estos tres lenguajes (HTML, CSS y JavaScript) son fundamentales para el desarrollo de sitios web modernos.
Css
Es un lenguaje de hojas de estilo que se utiliza para controlar la apariencia y el diseño visual de las páginas web, permitiendo separar la estructura del contenido de su presentación visual. Junto con HTML y JavaScript, CSS es una parte fundamental en el desarrollo de sitios web modernos y responsivos.
